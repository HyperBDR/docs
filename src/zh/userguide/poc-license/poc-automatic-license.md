# POC授权申请指南

1. **收到 POC 授权邮件**

    打开 POC 授权通知邮件，记录邮件中的 **项目 ID**，并点击【立即激活 POC】按钮，跳转至授权激活页面。

   ![](./images/poc-license-application-guide-1.png)

2. **授权激活页面**
   
   **2.1 输入项目ID**  
   在「测试项目 ID」输入框中填入邮件内的项目 ID，系统将自动获取对应的企业负责人邮箱。

   ::: tip 💡 提示
   请务必核实邮箱地址是否正确，确认无误后再执行后续操作。
   :::

   **2.2 点击 [获取验证码]**  
   点击【获取验证码】，并联系企业负责人获取对应的验证码。

   **2.3 添加额外的收件人邮箱**  
   如需将激活码邮件发送至其他邮箱，可添加新的接收邮箱地址。

   **2.4 输入POC授权数量**  
   填写本次申请的 **POC 授权数量**。

   **2.5 填入POC注册码**  
   填入 **POC 环境注册码**。

   ::: tip 💡 提示
   若需申请 **容灾** 或 **回切授权**，页面中将自动显示相应的回切注册码输入框。
   :::

   **2.6 点击 [获取激活码]**  
   ::: tip 💡 Tip
   对于单机版产品，需要从授权通知邮件中获取激活码，并手动复制到产品中完成激活。  
   对于 SaaS 产品，授权通常会自动完成激活；如果未自动激活，也可以从授权通知邮件中获取激活码，手动输入到 SaaS 产品中完成激活。
   :::

   ![](./images/poc-license-application-guide-2.png)
